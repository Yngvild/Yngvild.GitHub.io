About this compendium

This compendium provides an introduction to demographic methods in life history theory, including R code for main calculations, and is developed for the course BIOS5112/9112 at the University of Oslo. The online version of the document includes ´shiny-apps´ to help you visualize and understand some of the results, and a navigation bar making it easy to navigate among sections.

Before the widespread availability of computers, demographic methods consisted largely of manual calculations using life tables. However, the current computational tools and software like R (R Development Core Team 2020) make life table calculations much easier, and have also made matrix population models widely accessible. Matrix population models represent a powerful toolbox for analyses of life history and responses to climate change, among many other applications. In this course we will mostly focus on age structure, but the matrix model methods you will learn are the same for other kinds of population structure as well - such as spatial heterogeneity or life history stages. For those who are interested in learning these methods more thoroughly, Caswell (2001) provides a detailed introduction to matrix models for biologists.

In this compendium you will get the R code (including examples) for calculating all of the life history variables. It is important that you focus on understanding the biological meaning. Take your time to try out the calculations in the examples on your own, and solve the provided exercises. Solutions to the exercises will be provided in a separate document later.

As part of this course you will write a computational essay, which will count 40\(\%\) towards the final grade. This compendium will provide the background and most of the code you will need for the first part of this essay, where you will do a demographic analysis for a selected species. The note is written in R Markdown, which is a notebook format that integrates executable R code, outputs of the R code (such as plots), text, images, etc. Your computational essay will be written with R Markdown as well.

The note is still work in progress - if you find any errors or have questions / suggestions, please send an email (yngvild.vindenes@ibv.uio.no).


Learning goals

After reading this note (and doing the weekly exercises of the course), you should be able to

  • define and explain all the terms highlighted in bold type.

  • describe the exponential model for population growth and its importance in ecology and evolutionary biology.

  • explain main survivorship patterns over age and how these are linked to life history.

  • define a life table and use it to calculate the net reproductive rate \(R_0\) and cohort generation time \(T_C\).

  • explain the difference between a dynamic and static life table.

  • define the Euler-Lotka equation for age-structured populations, explain its main components and importance to life history theory.

  • discuss the advantages and disadvantages of using \(\lambda\) and \(R_0\) as fitness measures.

  • define a projection matrix for age- or stage-structured populations.

  • explain the main differences between a pre- and post-reproductive census.

  • draw a life cycle graph and explain how it is related to the projection matrix.

  • calculate asymptotic parameters from a matrix model (using projection or eigen analysis), and explain their biological meaning.

  • calculate sensitivities of \(\lambda\) with respect to projection matrix elements, and explain their meaning and some applications.

  • do a simple LTRE analysis for an age structured population in two different environments.

  • discuss some key properties of population growth in stochastic environments.

  • mention different life history strategies for variable environments.
